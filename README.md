# Pandora API

A REST API to serve as a backend for the [Pandora Mobile App](https://github.com/black-purple/pandora), made using Spring Boot.

## Dependencies
+ MySQL Driver
+ Spring Web
+ Spring Data JPA
+ Spring Security
+ Lambok
